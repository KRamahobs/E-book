# Net Ninja Pro - The Book

A responsive landing page built with **Bootstrap 5** to showcase the "Net Ninja Pro - The Book".  
Designed to highlight the book, pricing plans, chapters overview, reviews, and contact options.

---

## Demo

Check out the live demo here:  
[Live Demo](https://your-live-demo-link.com)

---

## Features

- Fully responsive layout for desktop and mobile
- Mobile-friendly navigation with collapsible menu
- Pricing plans section with highlighted "Most Popular" plan
- Accordion to showcase book chapters
- Book reviews section with ratings
- Contact form and email registration modal
- Offcanvas sidebar for additional books
- Built using **Bootstrap 5**, **Bootstrap Icons**, and minimal custom CSS
